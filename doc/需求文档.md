## 2. 产品定位
Leeson-ssh 是一款**现代化、轻量、跨平台的 SSH 远程连接工具**，主打简洁高效的服务器管理体验，通过书签、多标签、命令笔记本、自定义快捷键提升日常运维效率，界面采用 Ant Design 暗色风格，兼顾美观与实用性。

---

## 3. 核心功能总览
- SSH 基础连接（密码 / 私钥）
- 服务器书签管理
- 多标签终端
- 命令笔记本（本地 TXT + Ctrl/Cmd+I 发送）
- 自定义命令快捷键
- Ant Design 现代暗色界面
- Windows + macOS 双平台兼容

---

## 4. 详细功能需求

### 4.1 SSH 连接功能（核心基础）
1. **支持标准 SSH 协议连接**
2. **连接方式**
   - IP / 域名 + 端口（默认 22）
   - 用户名认证
3. **认证方式**
   - 密码登录
   - 私钥登录（支持 .pem / id_rsa 等格式）
4. **连接状态**
   - 未连接
   - 连接中
   - 已连接
   - 连接失败/超时/断开
5. **异常处理**
   - 连接超时提示
   - 认证失败提示
   - 网络异常自动提示
6. **快速连接**
   - 无需保存书签，可临时填写信息直连服务器

---

### 4.2 书签管理
1. 保存 SSH 连接信息为书签：名称、主机、端口、用户名、认证方式、备注
2. 支持书签分组
3. 支持书签搜索（按名称/IP 模糊匹配）
4. 左键单击书签 → 自动新建标签并建立 SSH 连接
5. 右键菜单：编辑、删除、复制信息、移动分组
6. 配置本地持久化，重启软件不丢失

---

### 4.3 多标签终端
1. 支持同时打开多个 SSH 终端标签
2. 标签标题可自定义重命名
3. 标签显示连接状态标识
4. 支持：新建标签 / 关闭标签 / 关闭其他标签 / 关闭全部
5. 终端支持复制、粘贴、清屏、自适应窗口大小
6. 支持终端分屏（横向/纵向）

---

### 4.4 命令笔记本
1. 打开本地任意 `.txt` 文件作为命令笔记
2. 实时编辑、自动保存
3. 光标所在行高亮
4. 发送当前行快捷键：
   - Windows：`Ctrl+I`
   - macOS：`Cmd+I`
5. 发送规则：
   - 自动去除首尾空格
   - 自动追加回车执行
   - 发送至当前激活终端
6. 面板可折叠、展开、独立悬浮

---

### 4.5 命令快捷键
1. 自定义键盘快捷键绑定常用命令
2. 支持组合键：Ctrl / Alt / Shift（Windows）；Cmd / Opt / Shift（macOS）
3. 可设置生效范围：全局 / 当前标签
4. 自动检测快捷键冲突
5. 支持导入导出快捷键配置

---

### 4.6 界面与主题
1. 采用 Ant Design 组件体系
2. 默认暗色主题
3. 主界面布局：
   - 左侧：书签面板
   - 顶部：终端标签栏
   - 中间：终端区域
   - 右侧：命令笔记本
4. 高 DPI / Retina 屏适配
5. 支持字体大小调整

---

## 5. 跨平台交互规则
| 操作 | Windows | macOS |
| --- | --- | --- |
| 发送命令笔记行 | Ctrl+I | Cmd+I |
| 复制 | Ctrl+C | Cmd+C |
| 粘贴 | Ctrl+V | Cmd+V |
| 新建标签 | Ctrl+T | Cmd+T |
| 关闭标签 | Ctrl+W | Cmd+W |
| 窗口控制 | 右上角 | 左上角（红绿灯） |
| 文件路径 | 盘符路径 | Unix 路径 |
| 敏感信息存储 | 本地加密 | 系统 Keychain |

---

## 6. 非功能需求
- 启动速度 ≤ 4s
- 标签切换流畅无卡顿
- SSH 连接稳定不掉线
- 支持 7×24h 后台运行不崩溃
- 密码、私钥仅本地存储，不上传云端
- 双平台体验一致

---

## 7. 验收标准
1. SSH 连接、密码/私钥认证正常
2. 书签增删改查及一键连接可用
3. 多标签终端正常切换、关闭、执行命令
4. 命令笔记本可打开 TXT，`Ctrl/Cmd+I` 可发送当前行
5. 自定义快捷键可正常绑定并执行
6. 界面为 Ant Design 暗色主题
7. Windows、macOS 均可正常安装、运行、卸载
